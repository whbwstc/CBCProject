## 作品简介
本系统是我本科毕业设计，仿照存不存寄存系统完成。
本系统致力于研究构建行李寄存的管理系统化解决办法，形成一套良好的系统化寄存服务方案。利用店铺空闲空间放置寄存柜来满足用户的寄存需求，可以解决用户在长途或者短暂出行的途中，轻装上阵，避免由于大包小包的行李所带来的困扰，尤其是在旅游风景区、火车站、飞机场等，便利人们的出行。当然，也能利用店铺空闲空间的资源，为店铺实行利益最大化。
